# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-08

### Added
- React entrypoint with `StyledQR` component.
- Presets: `default`, `rounded`, `circle`, `diamond`.
- Rounded contour-based rendering for smoother module shapes.
- Playground and integration docs for shadcn composition patterns.

### Changed
- Project rebranded from `prettyqr` to `styledqr`.
